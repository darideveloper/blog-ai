# Install

## Requirements
* [note js](https://nodejs.org/es) >= 16.15

## Clone repo

```bash
$ git clone https://github.com/darideveloper/dari-dev-blog.git
```

## Install dependencies

```bash
$ npm install
```

# Run

## Run in local
```bash
$ npm run dev
```

# Build

## Convert project to static files
```bash
$ npm run build
```
